snmpget, snmpgetnext - retrieve information an snmp-capable device using single reques

snmpwalk, snmpatable, snmpdelta - for multiple request

snmpset - manipulate configuration on an SNMP-capable device

snmpfd, snmpnetstat, snmpstatus - fixed collection of information

snmptranslate - convert between numerical and textual form of MIB OIDs, and
display content and structure.

tkmib - graphic MIB browser.

snmptrapd - deamon application. selected notification can be logged, forwarded
to another SNMP manager system or passed to an external application.


snmpd - an extensible agent for responding to SNMP queries for management information. This included support for wide range of MIB information modules, and can
be extended using dynamivally loaded modules, external scripts and commands, and both the SNMP  multiplexing (SMUX) and Agent Extensibility (AgentX) protocols.
